# TP3 - Autoencodeurs

## Description

Ce projet a pour objectif de mettre en œuvre un autoencodeur à travers trois exemples : 
1. Un exemple de base.
2. Le débruitage d'images.
3. La détection d'anomalies.

## Technologies

- Python 3
- TensorFlow
- Keras
- NumPy
- Pandas
- Matplotlib
- Seaborn
- scikit-learn
